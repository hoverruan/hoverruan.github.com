用Octopress创建我的Blog

Dec 14th, 2012

今年 ThoughtWorks 开源大奖得主是 Frank 的作者 Pete Hodgson, 在浏览其github主页时发现了他的博客Repository Being Agile，顿时眼前一亮，这正是我一直寻找的在Github上面搭建博客系统的理想方案！

这是基于 Octopress 的一个博客系统，是对著名的 Github 博客系统 Jekyll 进行二次开发的衍生产品，对手机访问进行了很大的优化，同时比 Jekyll 看起来更像是一个传统的博客。前者的风格看起来实在有点太简单了，需要花一些功夫才能在 Jekyll 的基础上配置一个看上去比较顺眼的风格的博客，因此一直迟迟没有动手。

GitHub Pages 是 Github 又一个简单而强大的功能，包含免费的 Hosting 服务和 Theme 功能。Github 自己也使用 Jekyll 作为他们网站大部分页面的管理方案。
